Terminal-style Portfolio
=======================

This is a small static "terminal" portfolio meant for hosting on Cloudflare Pages (free).

Files
- index.html — entry
- styles.css — presentation
- script.js — commands and typing effect

Deploy to Cloudflare Pages (static site)

1. Push this repository to GitHub.
2. In Cloudflare Pages, create a new project and connect to the repo.
3. Build command: leave blank. Build directory: `/` (root).
4. Save and deploy.

Customization

- Edit `script.js` 
- Style via `styles.css`.
